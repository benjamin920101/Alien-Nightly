## ⚙️ 核心管理器架構

```
Alien (主入口)
├── ModuleManager    — 模組載入與管理
├── CommandManager   — 命令系統（預設前綴：;）
├── ConfigManager    — 設定儲存與載入
├── EventBus         — 事件總線（高效 Lambda 事件系統）
├── RotationManager  — 旋轉角度管理
├── HoleManager      — 洞口偵測
├── InventoryManager — 物品欄管理
├── MovementManager  — 移動管理
├── NetworkManager   — 網路封包管理
├── ShaderManager    — Shader 管理（基於 Satin）
├── BlurManager      — 模糊效果管理
├── FriendManager    — 好友清單管理
├── XrayManager      — X-Ray 設定
├── TradeManager     — 交易管理
├── PopManager       — 圖騰爆炸偵測
├── BreakManager     — 破壞管理
├── ServerManager    — 伺服器資訊
└── ThreadManager    — 多執行緒管理
```
